# Objetivo
Utilizar la DI para consumir un API de perros

# Descripcion
Este proyecto tiene como objetivo consumir un API de perros y mostrarlos en una lista.

# Arranque
Para arrancar el proyecto es necesario tener un servidor local que provea la API de perros.
Se puede descargar el proyecto desde el siguiente repositorio:
https://github.com/rubenleisfp/woof_api
